📌 Qu’est ce que c’est React ?

React is a free and open-source front-end JavaScript library for building user interfaces based on UI components. It is maintained by Meta and a community of individual developers and companies.

React est une bibliothèque JavaScript conçue pour construire des interfaces utilisateurs. C’est un Single-page-app. Il permet de construire des composants réutilisables. Pour le mobil (react native). React n’est pas un framework, mais une librerie.

C’est normal de se sentir overwhelmed quand on apprend le code. La meilleur option c’est d’apprendre une seule chose et s’appuyer vraiment dessus. C’est mieux d’être expert dans une seule chose que mediocre dans une centaine.

DOM

The Document Object Model is a cross-platform and language-independent interface that treats an XML or HTML document as a tree structure wherein each node is an object representing a part of the document. The DOM represents a document with a logical tree.

Est une répresentation des objects HTML. C’est un peu le pont entre le code et l’ UI (interface utilisateur).

JSX

React ne peut pas render (ReactDOM.render) du vrai JavaScript. JSX est une extension syntaxique pour JavaScript qui permet d’ecrire votre IU dans une syntaxe de type HTML. C’est une surcouche du JS qui permet de melanger le langage le JS et le langage de valise. Il n’ya pas besoin d’apprendre des nouveaux symbols ou des syntaxes.

Babel

Est un outil qui permet de prendre tout les langages de et le code dans des versios lisibles par le navigateur
